Paragraph

Use <p></p> to create a paragraph of text in HTML. The browser will add a space before and after the content of the paragraph. For example:

This is a paragraph. Lorem ipsum dolor sit, amet consectetur adipisicing elit. Suscipit velit corporis reiciendis impedit earum, dolore provident aut numquam doloremque quos saepe, assumenda deserunt quisquam. Incidunt pariatur voluptatibus quidem natus omnis!

Highlighting Text

Use <mark></mark> to highlight the text for reference.

Form Tag

Use <form></form> to create a form that allow user to input different type of data. Each input must has its label.
